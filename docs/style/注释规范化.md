# 注释规范化

## 一、导入库

```c
/*=============== #include "driver/spi_master.h" ===============*/
```

## 二、文件头注释

​	**位于每个.c与.h文件的头部，用于描述文件的基本信息。**

```c
/**************************************************
 * @file    bsp_driver_spi.c
 * @brief   板级spi外设驱动
 * @author  xLumina
 * @version V1.0.0
 * @date    2026-4-8
 *
 * @par History:
 *   - V1.0.0 | 2026-4-8 | xLumina | 初始版本创建
 *
 * @par Function List:
 *      - bsp_spi_init()        : 初始化spi外设
 * 
 * 
 **************************************************/
```

## 三、函数注释

​	**函数声明处（通常在头文件中）的注释应详细描述其功能、参数、返回值和用法。**

```c
/**
 * @brief   初始化spi外设
 *
 * @param   NULL
 *
 * @return  NULL
 *
 * @note    板级外设初始化函数，只能由bsp调用
 */
```

## 四、数据结构——结构体、枚举、数组注释

```c
/**
 * @brief   通信数据包结构
 * @details 用于UART通信协议的帧结构定义，包含帧头和载荷。
 */
typedef struct {
    uint8_t  header;      	/* 帧头固定为 0xAA */
    uint8_t  length;      	/* 数据载荷长度 */
    uint8_t  data;    		/* 数据载荷内容 */
    uint8_t  checksum;    	/* 校验和: 前面所有字节累加和 */
} CommPacket_t;
```

## 五、代码块注释

```c
/* 遍历查找下一个空闲的任务槽 */
```

## 六、变量与宏定义注释

```c
/*************** #define USE_SPI 1 ***************/
```
